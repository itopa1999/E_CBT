  1. Exam Permission
    it is use for controlling whether student(s) starts the exam course(s) or not, If Exam Permission is ON. it means that student(s) can start the exam course(s), If Exam Permission is OFF it means that student(s) can not start the exam course(s). The Default Exam Permission for all courses is OFF
  2. Exam Submit Control
    it is use for auto submitting the student that are currently writing course(s) exam. if he Exam Submit Control is ON, it means that all student(s) can still write the ongoing course(s) exam, if the Exam Submit Control is OFF, that means that all the student(s) currently writing the ongoing course(s) exam will be automatically submitted and Auto grade. Default for all course(s) is ON.
  3. Student Permission
    it is use for activate and deactivate student(s) to or from writing exam. if Student Permission is ON, it means that student(s) can write exam, if the Student Permission is OFF, it means that student(s) cannot write or enter into the exam mode. Default for all student(s) is ON.
  4. Result Permission
    it is use to control whether result(s) should be display to student or not. if Result Permission is ON, it means that all students can view their scores, if Result Permission is OFF, it means that all students cannot view their scores. Default for this Result Permission is OFF.
